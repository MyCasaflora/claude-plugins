# Tätigkeitskategorisierung

Pflichtschritt VOR jeder Lizenz- und Registerprüfung. Die Tätigkeit muss einer
Kategorie zugeordnet werden, um die richtige Rechtsgrundlage zu wählen.

---

## Kategorisierungstabelle

| Kategorie | Typische Tätigkeiten | Relevante Rechtsgrundlage |
|---|---|---|
| **Immobilienvermittlung** | Makler, Bauträger, Baubetreuer, Wohnimmobilienverwalter | §34c GewO |
| **Finanzanlagevermittlung** | Fonds, Vermögensanlagen, geschlossene Beteiligungen | §34f GewO |
| **Versicherungsvermittlung** | Versicherungsberatung, -vermittlung, LV-Rückkauf | §34d GewO |
| **Immobiliardarlehen** | Baukreditvermittlung, Hypothekenberatung | §34i GewO |
| **Anlageberatung (reguliert)** | Wertpapierberatung, Portfolioverwaltung | §32 KWG / WpIG |
| **Krypto-Dienstleistungen** | Krypto-Beratung, -Vermittlung, -Verwahrung | MiCAR, §32 KWG |
| **Zahlungsdienste** | Debitkarten, Zahlungsabwicklung, E-Money | ZAG §10 |
| **Kapitalmarktprodukte** | Prospektpflichtige Anlagen, Crowdinvesting | VermAnlG §6, WpPG |
| **Steuerberatung** | Steuerliche Gestaltungsberatung | StBerG §5 |
| **Rechtsberatung** | Rechtliche Empfehlungen, Vertragsgestaltung | RDG §3 |
| **Gesundheitsprodukte** | Heilversprechen, Nahrungsergänzung, Medizinprodukte | HWG, MPG/MDR |

---

## Kontrastanalyse-Regel

Wenn auf derselben Plattform externe Akteure MIT Lizenzangaben auftreten
(z.B. "Makler nach §34d GewO"), die Hauptverantwortlichen aber KEINE Lizenz ausweisen:

→ Das ist ein besonders starkes Indiz: Die Betreiber kennen das Konzept der Lizenzpflicht
  und wenden es selektiv an.

**Prüffrage:** Werden auf derselben Website für andere Personen Lizenzangaben gemacht,
die für die Hauptakteure fehlen?
